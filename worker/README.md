部署时填好 `UPLOAD_TOKEN`，完成后把同一个值填进 AstrBot WebUI 的“网页日报设置”。

构建时会自动从 `SXP-Simon/astrbot_plugin_qq_group_daily_analysis@main` 拉取模板；如需改来源，可设置 `TEMPLATE_SOURCE_REPO`、`TEMPLATE_SOURCE_REF`、`TEMPLATE_SOURCE_PATH`。
